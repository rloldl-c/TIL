## 델타 탐색
- 이차원 리스트에서 완전 탐색 방법
- 각 지점에서 상하좌우에 위치한 다른 지점을 조회하거나 이동
```python
# 델타값 정의(상하좌우)
dx = [-1, 1, 0, 0]
dt = [0, 0, -1, 1]

# 이차원 리스트 순회
for x in range(n):
    for y in range(m):
        # 정의된 델타값으로 상하좌우 이동
        for i in range(4)
            nx = x + dx[i]
            ny = y + dy[i]

            # 행렬 범위를 벗어나지 않도록
            if 0 <= nx < n and 0 <= ny < m:
                x = nx
                y = ny
```
