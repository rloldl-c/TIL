# 오늘 한 것
- SQL
    - `SELECT field_name FROM table_name`
    - 기본적인 사칙연산 가능
    - 하나 이상의 필드를 기준으로 정렬 가능
        - 기준이 하나 이상일 때는 우선 하나의 기준으로 정렬한 후 그 상태에서 두번째 기준으로 정렬을 수행
        - 값이 겹치는 필드만 두번째 정렬이 수행되는 것
